# ☁️ 阿里百炼平台接入指南

> 本文档详细介绍如何将「每日气运值 MCP 服务」接入阿里百炼平台，让百炼的智能体能够调用气运值计算功能。
>
> 最后更新：2026-04-14

---

## 📌 两种接入方式对比

你的 MCP 服务已经部署在阿里云服务器上（`http://47.99.175.242:8890/sse`），接入百炼有 **两种方式**：

| 方式 | 说明 | 适合场景 | 你是否适用 |
|------|------|---------|-----------|
| **方式一：使用脚本部署** | 在百炼平台内配置远程 SSE 地址 | ✅ **最适合你**，直接复用已有服务 | ⭐ 推荐 |
| **方式二：从 AI 网关导入** | 通过阿里云 AI 网关中转 | 有域名、需高可用、企业级场景 | 备选 |

> ❌ **不需要**「插件」方式和「阿里云 OpenAPI」方式——插件是百炼官方内置的 MCP，OpenAPI 是操作阿里云产品的，都跟你的自定义气运值服务无关。

---

## 方式一：使用脚本部署（推荐 ⭐）

这种方式直接把你的远程 SSE 服务地址告诉百炼平台，百炼会自动连接。

### 第1步：进入百炼 MCP 管理页面

1. 打开 [阿里云百炼控制台](https://bailian.console.aliyun.com/?tab=mcp#/mcp-market)
2. 点击页面右上角的 **「+」** 按钮

### 第2步：选择创建方式

- 在弹出的选项中，选择 **「使用脚本部署」**
- 点击 **「部署服务」**

### 第3步：填写基本信息

| 配置项 | 填写内容 |
|--------|---------|
| **服务名称** | `每日气运值` （自定义，不影响功能） |
| **服务描述** | `基于传统易学算法，计算每日气运值分数（60~100分），提供五行分析和开运建议` |
| **安装方式** | 选择 **`http`** （远程服务） |

### 第4步：填写 MCP 服务配置（⭐ 最关键！）

在「MCP 服务配置」输入框中，填入以下 JSON：

```json
{
  "mcpServers": {
    "daily-fortune": {
      "url": "http://47.99.175.242:8890/sse",
      "type": "sse"
    }
  }
}
```

> ⚠️ **注意**：
> - 把 `47.99.175.242` 替换成你自己的服务器公网 IP
> - 如果你配了 Nginx + 域名 + HTTPS，改成 `https://fortune.yourdomain.com/sse`
> - 地址末尾必须有 `/sse`
> - **不要加 headers，你的服务不需要认证**

### 第5步：提交部署

点击 **「部署服务」** 或 **「提交」**，等待部署完成。

### 第6步：验证服务

1. 部署成功后，在「MCP 管理 → 自定义服务」中找到 `每日气运值`
2. 点击该服务，进入详情页
3. 在「工具」页面，应该能看到 3 个工具：
   - ✅ `calculate_fortune` — 计算气运值
   - ✅ `get_wuxing_info` — 查询五行信息
   - ✅ `get_ganzhi_info` — 查询天干地支信息
4. 点击 **「运行」** 按钮，选择 `calculate_fortune`，输入测试参数：
   - `birth_date`: `1990-06-15`
   - 点击执行，确认返回了气运值 JSON

### 第7步：绑定到智能体

1. 前往 [百炼应用管理](https://bailian.console.aliyun.com/?tab=app#/app-center)
2. 点击 **「创建应用」** → 选择 **「智能体应用」** → **「立即创建」**
3. **选择模型**（⚠️ 重要：必须选 **Plus 模型**，Max 模型不支持 MCP！）
4. 在配置页面中找到 **「MCP 服务」** 配置区
5. 勾选刚才创建的 `每日气运值` 服务（最多可添加 5 个）
6. 在系统提示词中填入（参考下方模板）
7. 点击 **「发布应用」**

### 第8步：系统提示词模板

在智能体的「系统提示词/人设」中，填入以下内容：

```
你是一位精通传统易学的气运大师。你可以根据用户的出生日期和时间，结合当日的天干地支和五行生克关系，计算每日气运值。

## 可用工具
- calculate_fortune：计算气运值（核心工具），需要传入 birth_date（格式 YYYY-MM-DD）
- get_wuxing_info：查询五行详细属性
- get_ganzhi_info：查询四柱八字信息

## 回答规则
1. 当用户询问运势、命理、今日运势等相关问题时，主动调用 calculate_fortune
2. 如果用户只提供了出生日期没有提供出生时间，直接用 birth_date 参数即可（不传 birth_time）
3. 根据返回的气运分数和详细分析，用生动有趣、通俗易懂的方式解读
4. 语气亲切、有温度，可以适当加入传统文化元素
5. 主动提供开运建议（幸运色、吉位等）

## 示例对话
用户：我今天运势怎么样？我是1990年6月15日出生的
你的操作：调用 calculate_fortune(birth_date="1990-06-15")，然后解读结果
```

## 方式二：从 AI 网关导入（备选）

如果你有域名，或者需要更高的可用性和安全管控，可以用 AI 网关方式。

### 前置条件

需要先在「AI 网关」中托管你的 MCP 服务。这需要：
- 一个已备案的域名
- 在 AI 网关中创建实例并配置后端指向你的 MCP 服务

### 操作步骤

1. 前往 [阿里云百炼 MCP 管理](https://bailian.console.aliyun.com/?tab=mcp#/mcp-market)
2. 点击 **「+」** → 选择 **「AI 网关」** → 点击 **「导入服务」**
3. 填写配置：

| 配置项 | 说明 |
|--------|------|
| **网关所在区域** | 选择你 AI 网关所在地域（如：华东1（杭州）） |
| **网关实例** | 选择对应的网关实例 |
| **选择MCP服务** | 选择你在网关中托管的 MCP 服务名称 |
| **选择接入点** | 系统自动生成 |

4. 点击 **「确认导入」**
5. 后续绑定智能体的步骤同方式一的第 7-8 步

> 💡 AI 网关方式的好处：支持 HTTPS、域名绑定、流量管控、访问日志等企业级功能。但配置更复杂，个人项目用方式一就够了。

## 🔧 MCP 配置 JSON 速查

百炼平台的「MCP 服务配置」JSON 中，`mcpServers` 支持以下格式：

### 远程 SSE 服务（你的情况）

```json
{
  "mcpServers": {
    "daily-fortune": {
      "url": "http://你的服务器IP:8890/sse",
      "type": "sse"
    }
  }
}
```

### 如果加了 HTTPS + 认证头

```json
{
  "mcpServers": {
    "daily-fortune": {
      "url": "https://fortune.yourdomain.com/sse",
      "type": "sse",
      "headers": {
        "Authorization": "Bearer your-token"
      }
    }
  }
}
```

### 本地 stdio 方式（在百炼 FC 上运行代码时用）

```json
{
  "mcpServers": {
    "daily-fortune": {
      "command": "uvx",
      "args": [
        "--with-editable",
        "./src",
        "daily-fortune-mcp",
        "--transport",
        "stdio"
      ],
      "type": "stdio"
    }
  }
}
```

## ❓ 常见问题

### Q1: 部署后「工具」页面看不到我的 3 个工具？

- 检查 JSON 配置中的 `url` 是否正确，末尾是否有 `/sse`
- 在服务器上执行 `curl -N --max-time 5 http://localhost:8890/sse` 确认服务正常
- 检查阿里云安全组是否放行了 8890 端口

### Q2: 智能体不调用 MCP 工具？

- ⚠️ **必须使用 Plus 模型**，Max 模型不支持 MCP
- 检查系统提示词中是否明确提到了要调用气运工具
- 智能体根据对话内容自动判断是否调用，提问中需要包含"运势""命理"等关键词

### Q3: 工具调用返回错误？

- 到百炼 MCP 管理页面，点击服务名称进入详情
- 查看调用日志和错误信息
- 同时在服务器上执行 `sudo docker logs --tail 50 daily-fortune-mcp` 查看服务端日志

### Q4: 「安装方式」选 http 还是 npx/uvx？

- 你的服务部署在自己的服务器上 → 选 **http**
- 你想把代码上传到百炼的函数计算上运行 → 选 **uvx**

### Q5: 四种创建方式到底怎么选？

| 方式 | 什么时候用 |
|------|-----------|
| **插件** | 不需要选，这是百炼官方内置的 MCP（如高德地图） |
| **脚本部署** ⭐ | **你用它**——自己开发/找开源 MCP，配置地址或上传代码 |
| **AI 网关** | 有域名 + 需要 HTTPS + 企业级管控 |
| **阿里云 OpenAPI** | 让 AI 操作阿里云产品（OSS、ECS等） |
